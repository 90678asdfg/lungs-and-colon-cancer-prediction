# lungs-and-colon-cancer-prediction
• Developed a deep learning-based classification system using transfer learning (ResNet50, VGG16, EfficientNet-B0) for histopathological image analysis; ResNet50 achieved the best performance with up to 98% validation accuracy.
• Implemented Grad-CAM for model interpretability, enabling visualization of critical regions influencing predictions and enhancing clinical trust
• Presented at IEEE IC-ICNS 2026, highlighting research on explainable AI for cancer diagnosis under the
guidance of Prof. Sunil Kumar Mohapatra; developed a Streamlit-based web application for real-time prediction
and visualization.
